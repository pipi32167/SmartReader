# 更新日志

## 1.0.5 - 2026-04-30

### 新功能
- Popup 自定义提示词区域自动检测上下文状态（网页全文 / 选中文字）
- 自定义提示词新增「附带页面内容」勾选框，默认勾选；反选时仅发送提示词
- 历史记录详情页新增「查看完整提示词」弹窗，支持查看完整提示词内容

## 1.0.4 - 2026-04-28

### 新功能
- Popup 自定义输入框内容缓存，关闭后重新打开可恢复

### 文档
- 新增中英文 README

## 1.0.3 - 2026-04-27

### 新功能
- 新增历史记录搜索功能
- 支持删除对话历史中的单条消息
- 新增 AI 回复重新生成按钮

## 1.0.2 - 2026-04-26

### 新功能
- 新增对话历史记录，使用 SQLite 持久化存储
- 支持多轮追问对话

- 设置页面支持 API 配置与提示词管理
