# 更新日志

## 1.0.10 - 2026-05-08

### 新功能
- 添加历史记录收藏功能
- 支持在历史列表和详情页收藏/取消收藏
- 支持筛选仅显示收藏记录
- 收藏记录默认排在列表顶部

### 修复
- 修复 GET_HISTORY_LIST 消息处理器未传递 favoritesOnly 筛选参数的问题

## 1.0.9 - 2026-05-08

### 重构
- 简化 PDF 处理流程，统一先转换为 Markdown 再发送给 AI
- 移除冗余的 OpenAI `file` 内容类型上传逻辑
- 移除 API 拒绝文件类型时回退到纯文本提取的逻辑

## 1.0.8 - 2026-05-07

### 文档
- 更新 README 中的 Chrome Web Store 插件链接

## 1.0.5 - 2026-04-30

### 新功能
- Popup 自定义提示词区域自动检测上下文状态（网页全文 / 选中文字）
- 自定义提示词新增「附带页面内容」勾选框，默认勾选；反选时仅发送提示词
- 历史记录详情页新增「查看完整提示词」弹窗，支持查看完整提示词内容

## 1.0.4 - 2026-04-28
